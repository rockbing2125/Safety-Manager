# Excel 参数模板格式说明

## 模板列顺序

从 v1.1.8 开始，Excel 参数模板新增了"系数"列，列顺序如下：

| 列序号 | 列名 | 说明 | 示例 |
|--------|------|------|------|
| 1 | 类别 | 参数分类 | 电压等级要求 |
| 2 | 参数 | 参数名称 | 额定电压 |
| 3 | 默认值 | 参数默认值 | 220V |
| 4 | 上限 | 参数上限值 | 250V |
| 5 | 下限 | 参数下限值 | 200V |
| 6 | 单位 | 参数单位 | V |
| 7 | **系数** | **参数系数（新增）** | 1.0 |
| 8 | 备注 | 备注说明或图片 | 可选 |

## 重要提示

1. **系数列为新增列**：位于"单位"列和"备注"列之间
2. **旧的 Excel 模板**：如果使用旧模板（7列），系统会自动识别，但建议更新为新格式（8列）
3. **系数列可选**：系数列可以为空，不影响其他功能
4. **图片支持**：备注列支持插入图片（使用 Excel 的 DISPIMG 函数）

## 数据库迁移

如果您已有旧数据，系统会自动添加 coefficient 字段，无需手动操作。

运行以下命令可手动迁移数据库：
```bash
python add_coefficient_column.py
```

## 示例

```
| 类别       | 参数     | 默认值 | 上限  | 下限  | 单位 | 系数 | 备注           |
|-----------|---------|-------|-------|-------|------|------|----------------|
| 电压要求   | 额定电压 | 220V  | 250V  | 200V  | V    | 1.0  | 标准工作电压    |
| 频率响应   | 工作频率 | 50Hz  | 52Hz  | 48Hz  | Hz   | 1.0  | 电网标准频率    |
```
